#ОПИСАНИЕ 
Приложение считытывает из файла url-адерсса. Кнопка (Прочитать файл с URL)
Приложение считает количество тегов "а" в html документах по адрессам. Кнопка (Начать парсинг)
Есть возможноть прерывания процусса подсчета. Кнопка (Отмена)
Url c наибольшим количеством искомых тегов веделяется зеленым цветом
Если URL не возможно открыть в поле для колическтва указывается проблема
#О файле со списком URL
В проекте есть пример файла exaple.txt
Url-адресса должны быть написанны по строчно.
Можно писать URL вида: название.домен www.название.домен https://www.название.домен http://www.название.домен
